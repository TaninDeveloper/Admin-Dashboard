🎀 Admin Dashboard

A modern and elegant administration panel designed for the Pinky Girl Shop. This dashboard allows store owners to manage their e-commerce operations with a seamless and aesthetic user experience.

✨ Key Features
- Dashboard Overview: Quick glance at store performance.
- Product Management: Full CRUD operations for categories and products.
- Order Tracking: Manage customer orders and their shipping status.
- Customer Relations: Detailed view and management of customer data.
- Marketing Tools: Create and manage discount coupons to boost sales.
- Feedback System: Review and manage customer comments.
- Store Settings: Customizable theme colors and shop configurations.

🛠️ Tech Stack
- Frontend: React.js
- Styling: Bootstrap 5 (Customized)
- Routing: React Router DOM
- Notifications: React Hot Toast
- Font: Vazir Font (Optimized for Persian typography)

🚀 Getting Started

1. Clone the repository:
   git clone https://github.com/TaninDeveloper/Admin-Dashboard.git
   
2. Install dependencies:
   npm install
   
3. Run the project:
   npm start
